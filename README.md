# OptiBuy - Real-Time Price Comparison System

OptiBuy is a smart web automation platform designed to fetch and compare product prices across multiple major e-commerce websites simultaneously. It eliminates the hassle of opening multiple tabs manually, providing users with a single, consolidated dashboard to find the best deals in real-time.



## 🚀 Features

- **Premium Netflix-Style UI:** Built with a highly responsive, modern dark theme featuring stylish product cards and smooth hover animations.
- **Dynamic Automation Backend:** Powered by Playwright to spin up headless browsers in the background, allowing it to navigate and handle complex JavaScript-heavy websites.
- **Multi-Platform Comparison:** Live synchronization across **Amazon**, **Flipkart**, **Myntra**, **Savana**, and **Purplle**.
- **Asynchronous Data Fetching:** Utilizes asynchronous programming to search multiple platforms concurrently, optimizing response time.



## 🛠️ Tech Stack

- **Frontend:** HTML5, Tailwind CSS, JavaScript (Fetch API)
- **Backend:** Python, Flask Framework
- **Automation & Scraping:** Playwright (Headless Browser Execution)
